⏸ HOW TO PAUSE THE SCANNER
To PAUSE:
1. Go to your seismic repo on GitHub
2. Click Settings → Secrets and variables → Actions
3. Click the Variables tab
4. Click New repository variable
5. Name: SCANNER_ENABLED · Value: false
To RESUME:
1. Go back to Settings → Secrets and variables → Actions → Variables
2. Click SCANNER_ENABLED and change value to true
💡 Quickest way to pause: Go to GitHub → Actions tab → find your scanner workflow → click the ⋯ menu → Disable workflow

Free Python library. No server needed. Run it on your own computer. Google may rate-limit if you scan too often.
⭐ Best value~$7/mo
pytrends + free cloud hosting

SerpAPI + Supabase + Vercel
SerpAPI ($50/mo) gives reliable data. Supabase stores history (free). Best for serious use or sharing with a team.
